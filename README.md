Example code for my Scalathon 2012 intro talk on lenses in Scala using [Shapeless](https://github.com/milessabin/shapeless/).

## How to run

```text
% git clone https://github.com/SethTisue/lens-examples
% cd lens-examples
% ./sbt
> test
...
> console
...
```
